## weather-app
Web application that showcases a weather dashboard.

## Setting up your local

# Gulp
1.  Install Gulp and gulp-sass package:
      npm install gulp gulp-sass --save-dev
2.  Run 'gulp' to watch and compile scss files
      gulp

# Sass
1. Install Sass
      npm install sass --save-dev

# Create directories
    Create the src/scss and dist/css folder inside the project root folder.

    Example:
    project-root/
    |__ src/
    |   |__ scss/
    |       |__ style.scss
    |__ dist/
    |   |__ css/
    |       |__ style.css
